Even though I had to get a ton of help from Dr. Sherman, i will be teaching you the steps to make a git hub repository and website.

Here is what you will need

  • A Git Hub Account
  • Terminal
  • Sublime Text

Create a repository

In your github account, you want to click in the top left the green new button. This will allow you to create your reporitory. When your click the new button, github will ask you to name your repository. MAKE SURE ITS PUBLIC. After you name the repository you will click the create repository button and type in all of the information listed to you. To keep your sanity, make sure you type in everything word by word.

Terminal and github

  1. open terminal and type next to your comupters user name cd and then drag your folder file into termainl. Once it is beside the cd then click enter.
  2. Type in echo "# (YOUR FOLDER NAME)" >> README.md then click enter
  3. Type git innit
  4. Type git commit -m "first commit" and hit enter
  5. Type git remote add origin with the HTTPS address listed and hit enter
  6. The final step to type is git push -u origin master
